Q&A

1. What are the three main differences between Web1, Web2, and Web3?

Web1 (1990-2004)

  • Read-only, static content
  • Centralized information flow from providers to users
  • Limited user interaction with mostly HTML-based websites

Web2 (2004-Present)

  • Read-write, dynamic content with user participation
  • Centralized platforms controlling user data
  • Social interaction and user-generated content

Web3 (Emerging)

  • Read-write-own, decentralized web
  • User ownership of data and digital assets
  • AI-driven semantic web with blockchain technology

2. Example: Searching for "side effects of medications"

Web1 Experience:

Static medical websites with basic HTML pages listing medication side effects. Information is:

  • Published by medical institutions only
  • Not interactive or personalized
  • Rarely updated static pages
  • Limited to text and basic images

Web2 Experience:

Interactive health platforms with user participation. Features include:

  • User reviews and experiences
  • Discussion forums and comments
  • Social media sharing of experiences
  • Regular updates based on user feedback

Web3 Experience:

Decentralized health data platforms. Offering:

  • Personal health data ownership
  • AI-powered personalized medicine insights
  • Verified patient experiences on blockchain
  • Direct connection with healthcare providers through smart contracts
